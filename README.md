# Challange-TakeBlip

### O desafio proposto era de um chatbot interativo com um carrossel de valores da take.


Criado primeiro chatbot utilizando a plataforma da Blip.

A ideia inicial proposta do carrossel e das mensagens iniciais foram atendidas:

<img src="https://raw.githubusercontent.com/mateusfilipe/Challange-TakeBlip/main/imgs/exemplo_1.png">

O chatbot envia as mensagens normalmente para início da conversa e envia o carrossel para o usuário.

Ao interagir com o botão no carrossel o chatbot envia a mensagem por padrão.

<img src="https://github.com/mateusfilipe/Challange-TakeBlip/blob/main/imgs/exemplo_2.png">

Após tal passo passei a ter problemas para a continuação do desafio.
